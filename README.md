JavaScript API 交互与数据操作：在已有页面基础上完成任务面板功能

这道题主要考查：

- JavaScript 基础语法
- DOM 获取与修改
- 事件绑定
- API 请求
- 接口数据渲染
- 基础数据筛选

本题会提前提供：

- 页面 HTML

- API 文档

你需要做的事情主要是：

- 写 JavaScript

## 如果这是模板仓库，你应该怎么开始

在 GitHub 页面点击：

```bash
git clone <你的仓库地址>
cd <你的仓库名>
```

### 第三步：重点修改 `scripts/main.js`

本题已经提供了基础页面，你主要需要完成：

- `scripts/main.js`

如果你愿意，也可以在不破坏页面结构的前提下微调 HTML 或 CSS，但评分重点在 JavaScript。

---

## 你的任务

你需要完成以下功能：

### 1. 获取任务列表
页面加载后，请调用 API 获取任务列表，并把数据渲染到页面中。

### 2. 刷新任务列表
点击“刷新任务”按钮时，需要重新请求接口并更新页面。

### 3. 提交新任务
表单提交时，需要把新任务发送给 API，并在成功后更新页面。

### 4. 状态筛选
当用户切换筛选框时，需要按状态展示任务：

- 全部
- 待开始
- 进行中
- 已完成

### 5. 错误提示
当接口请求失败时，需要在页面上给出基础提示。
