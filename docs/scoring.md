# 评分细则（面向 agent）

总分 100 分。

## 评分原则

1. 这道题面向初学者，不按资深前端标准打分。
2. 优先看功能是否完成，再看代码是否清楚。
3. 只要逻辑基本正确，就不要因为写法不够漂亮而给过低分。
4. 如果学生 README 能说清楚自己的思路，可以适当正向考虑。

## 一、DOM 获取与事件绑定，20 分
- 能正确获取核心元素，10 分
- 能正确绑定加载、点击、提交、筛选事件，10 分

## 二、GET 接口请求与列表渲染，25 分
- 能正确请求任务列表，10 分
- 能把数据渲染到页面，10 分
- 渲染结果基本正确，5 分

## 三、POST 接口提交，20 分
- 能正确获取表单数据，10 分
- 能正确发送请求并处理成功结果，10 分

## 四、数据处理能力，15 分
- 能按状态筛选任务，10 分
- 能基本保持页面数据更新正确，5 分

## 五、错误处理与用户提示，10 分
- 请求失败时有提示，5 分
- 提交成功或失败有基础反馈，5 分

## 六、代码质量与 README，10 分
- JavaScript 结构基本清楚，5 分
- README 有基本说明，5 分
