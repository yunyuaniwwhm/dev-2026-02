# 项目名称

## 项目简介

这是一个使用 JavaScript 完成 API 交互与数据渲染的任务面板页面。

## 我完成的功能

- 获取任务列表
- 渲染任务列表
- 提交新任务
- 按状态筛选任务
- 错误提示

## 我的实现思路

### 1. 页面加载

我在页面加载后请求了任务列表接口。

### 2. 获取和渲染数据

我把接口返回的数据渲染到了任务列表区域。

### 3. 提交任务

我在表单提交时获取输入内容，并发送 POST 请求。

### 4. 状态筛选

我通过下拉框筛选不同状态的任务。
